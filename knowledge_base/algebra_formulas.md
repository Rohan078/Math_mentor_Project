# Algebra – Key Formulas and Identities

## Quadratic equations
- For ax² + bx + c = 0, the roots are: x = (-b ± √(b² - 4ac)) / (2a).
- Discriminant D = b² - 4ac: if D > 0 two distinct real roots; D = 0 one repeated root; D < 0 no real roots.
- Sum of roots = -b/a, product of roots = c/a.

## Polynomial identities
- (a + b)² = a² + 2ab + b²
- (a - b)² = a² - 2ab + b²
- a² - b² = (a + b)(a - b)
- (a + b)³ = a³ + 3a²b + 3ab² + b³
- (a - b)³ = a³ - 3a²b + 3ab² - b³
- a³ + b³ = (a + b)(a² - ab + b²)
- a³ - b³ = (a - b)(a² + ab + b²)

## Inequalities
- AM–GM: (a + b)/2 ≥ √(ab) for a, b ≥ 0; equality when a = b.
- For |x| < a we have -a < x < a.
- |a + b| ≤ |a| + |b| (triangle inequality).

## Domain constraints (algebra)
- Denominators must not be zero.
- Expressions under √ must be ≥ 0 when working in reals.
- log base and argument must be positive; argument of log must be > 0.
