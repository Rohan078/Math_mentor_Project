# Calculus – Simple Optimization

## Steps for word problems
1. Draw figure if helpful; assign variables.
2. Write objective function Q to maximize or minimize.
3. Use constraints to express Q as function of one variable and state domain.
4. Differentiate and find critical points (Q' = 0 or Q' undefined).
5. Check endpoints of domain and critical points; pick global max or min.
6. Answer the question with units.

## Second derivative test
- At critical point c: if Q''(c) < 0 then local max; if Q''(c) > 0 then local min; if Q''(c) = 0 test is inconclusive.

## Domain constraints
- Lengths, areas, volumes must be positive.
- Variables often bounded by geometry (e.g. 0 < x < side length).
- Always state the domain before differentiating.

## Common mistakes
- Forgetting to check boundary values.
- Maximizing when problem asks for minimum (or vice versa).
- Wrong units or forgetting to substitute back to get the requested quantity.
