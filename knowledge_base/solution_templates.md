# Solution Templates – How to Present Step-by-Step Solutions

## General structure
1. Restate the problem clearly.
2. List given data and what is to be found.
3. State approach (e.g. “We use the quadratic formula”).
4. Show steps with brief justification (e.g. “Using identity (a+b)² = ...”).
5. Check: units, domain, special cases.
6. Final answer in a box or bold.

## Quadratic equation template
- Write equation in form ax² + bx + c = 0.
- Identify a, b, c; compute D = b² - 4ac.
- If D ≥ 0: x = (-b ± √D)/(2a). If D < 0: no real roots.
- Simplify and list roots.

## Probability template
- Define sample space or events clearly.
- Write formula used (e.g. P(A|B) = P(A∩B)/P(B)).
- Substitute numbers; simplify.
- Ensure 0 ≤ probability ≤ 1.

## Optimization template
- Quantity to optimize; constraint(s); express as f(x); domain of x.
- f'(x) = 0 ⇒ critical point(s).
- Compare f at critical points and endpoints; state max or min.
- Answer with units.
