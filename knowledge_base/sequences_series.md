# Sequences and Series (JEE-Relevant)

## Arithmetic progression (AP)
- a_n = a + (n-1)d. Sum to n terms: S_n = n/2 (first + last) = n/2 (2a + (n-1)d).

## Geometric progression (GP)
- a_n = a r^(n-1). Sum to n: S_n = a(1 - r^n)/(1 - r) when r ≠ 1.
- Infinite sum: a/(1 - r) when |r| < 1.

## Summation formulas
- Σk = n(n+1)/2, Σk² = n(n+1)(2n+1)/6, Σk³ = (n(n+1)/2)².

## Domain/constraints
- For GP, common ratio r ≠ 1 when using finite sum formula; |r| < 1 for infinite sum.
- n is positive integer for term count and sums.
