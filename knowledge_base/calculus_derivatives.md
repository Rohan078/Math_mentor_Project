# Calculus – Derivatives

## Definition
- f'(x) = lim_{h→0} (f(x+h) - f(x))/h. Slope of tangent line at x.

## Rules
- (u ± v)' = u' ± v'; (cu)' = c u'; (uv)' = u'v + uv'; (u/v)' = (u'v - uv')/v².
- Chain rule: (f(g(x)))' = f'(g(x)) g'(x).

## Standard derivatives
- d/dx x^n = n x^(n-1). d/dx e^x = e^x. d/dx ln|x| = 1/x.
- d/dx sin x = cos x, d/dx cos x = -sin x, d/dx tan x = sec² x.

## Applications
- Tangent line at x = a: y - f(a) = f'(a)(x - a).
- Critical points: f'(x) = 0 or f' undefined. Use first or second derivative test for local max/min.
- Second derivative: f'' > 0 ⇒ concave up, f'' < 0 ⇒ concave down; f''(c) = 0 and sign change ⇒ inflection.

## Optimization (JEE-style)
- Identify quantity to maximize/minimize; express as function of one variable; find domain; find critical points; compare values at critical points and boundaries.
