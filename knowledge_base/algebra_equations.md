# Algebra – Linear and Quadratic Equations

## Linear equations in one variable
- Standard form: ax + b = 0 ⇒ x = -b/a, provided a ≠ 0.
- Always check that the solution does not make any denominator zero in the original problem.

## Systems of linear equations (2×2)
- Substitution: express one variable from one equation and substitute into the other.
- Elimination: multiply equations to align coefficients and add/subtract to eliminate one variable.
- Consistency: unique solution (lines intersect), no solution (parallel), infinitely many (same line).

## Quadratic equations – solution methods
1. Factorization: write ax² + bx + c = a(x - α)(x - β) and use zero-product property.
2. Completing the square: rewrite as (x + p)² = q and take square roots (remember ±).
3. Quadratic formula: x = (-b ± √(b² - 4ac)) / (2a).

## Common mistakes
- Forgetting ± when taking square roots.
- Dividing by an expression that could be zero without checking.
- Sign errors when expanding (a - b)² or (a + b)(a - b).
