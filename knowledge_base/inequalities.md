# Inequalities (Algebra and JEE)

## Linear inequalities
- Same rules as equations for addition; multiplying/dividing by negative reverses inequality.
- Represent solution on number line or as interval.

## Quadratic inequalities
- Solve f(x) = 0 to get roots; determine sign of f in intervals. f(x) > 0 or f(x) < 0 in each interval.

## AM–GM–HM
- For positive a, b: (a+b)/2 ≥ √(ab) ≥ 2ab/(a+b). Equality when a = b.

## Absolute value
- |x| < a ⇔ -a < x < a. |x| > a ⇔ x < -a or x > a.
- |a - b| = distance between a and b on number line.

## Common mistakes
- Squaring both sides can introduce extraneous solutions; check in original.
- Dividing by a variable that could be negative without splitting cases.
